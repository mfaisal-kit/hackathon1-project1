## Specification Details

### Objectives Defined
- Modernize UI using Docusaurus v2 theming and layout features

### Technical Requirements
- Use native Docusaurus v2 theming (Infima, custom CSS, layout components)
- Maintain accessibility (WCAG-friendly colors and contrast)

### Constraints and Standards
- Tech stack: Docusaurus only

### Success Criteria

- Clean, consistent UI across all modules
- Book builds successfully with updated UI
- No broken links or sidebar issues
